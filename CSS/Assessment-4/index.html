Block elements take the full width of their parent so next element starts in the new line. We can also specify height and width to the block element.

Some examples of block elements are:

Div is a block element.

Paragraph is a block element.

h1-h6 are block elements.

section is a block element.
Table is a block element.
hr is a block element.
form is a block element.
    ol and li both are block elements.

  2. two

Inline elements only take width as their necessity. It doesnot start on a new line.

a tag is an inline element. span is an inline element. Input is an inline element. Strong is an inline element. Small is an inline element. img is an inline element. Map is an inline element.
