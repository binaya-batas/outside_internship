width: max(50%, 300px);

max() function: This function takes two arguments and applies whichever is maximum. We can see this by resizing the red-bordered outer box. In this case, the box either is 300px width or 50% width of the parent div, whichever is maximum.


width: min(50%, 300px)

min() function: This function takes two arguments and applies whichever is minimum. We can see this by resizing the red-bordered outer box. In this case, the box either is 300px width or 50% width of the parent div, whichever is minimum.


width: clamp(300px, 50%, 500px)

clamp() function: This function takes three arguments. This function keeps the width of the box between 300px and 500px and 50% of the parent's width if it is between first and last arguments.
